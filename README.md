# Fraud Detection MLOps Pipeline

This project implements a small end-to-end Machine Learning pipeline for detecting fraudulent transactions using both supervised and anomaly detection approaches.

The objective of the project is to demonstrate practical MLOps concepts such as:

- Feature engineering pipelines
- Model training and evaluation
- Artifact persistence
- Dependency versioning
- CI integration
- Unit testing

---

# Models

Two models were implemented:

### Random Forest (Supervised)

Used to classify transactions as fraudulent or legitimate.

### Isolation Forest (Unsupervised)

Used to detect anomalous transactions without labels.

This allows comparison between supervised learning and anomaly detection approaches.
